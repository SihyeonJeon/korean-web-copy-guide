# korean-web-copy-guide

AI가 생성하는 한국어 제품 페이지, SaaS 페이지, 대시보드, 포트폴리오 화면을 위한 웹 문구 규칙.

핵심 파일:

- [`KOREAN_WEB_COPY.md`](KOREAN_WEB_COPY.md): 한국어 문구, 문장 구조, 줄바꿈, 단어 선택, 금지 패턴

## 사용

```text
@KOREAN_WEB_COPY.md를 한국어 웹 문구 기준으로 사용한다.
페이지가 사용자 요청을 충족한다고 말하지 않는다.
보편적인 한국어 제품 문구, 의미 단위 줄바꿈, 관찰 가능한 사실만 사용한다.
```

## 경계

- 참고 기업의 공식 가이드 아님
- 재사용 가능한 브랜드 문구 모음 아님
- 출처의 슬로건, 제목, CTA, 고객 주장, 로고, 스크린샷, 제품명 복사 금지
- 새 페이지 문구는 해당 제품의 사실, 대상, 증거, 제약만 사용

## 조사 대상

공개된 한국어 제품, 엔터프라이즈, 클라우드, 핀테크, SaaS 페이지를 관찰했다.

- Toss
- Kakao Business
- Channel Talk
- Airbridge
- NAVER Cloud Platform
- Samsung SDS
- SK AX
